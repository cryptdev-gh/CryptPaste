# CRYPTPASTE

**A open-source pastebin alternative**

### Features

- [-] Simple and easy to use
- [X] Password protected pastes
- [X] Admin Panel
- [/] API (in progress)

### Installation

1. Clone repo
2. Customize your instance in the config.json
3. Run the script.sh (for developpement the docker compose is soon!)
4. have fun


### Requirements

- Python 3.x
- Flask
- SQLAlchemy
- A computer 
- A brain
