# CRYPTPASTE

**A open-source pastebin alternative**

### Features

- [-] Simple and easy to use
- [X] Password protected pastes
- [X] Admin Panel
- [/] API (in progress)

### Installation

1. Clone the repository
2. Install the requirements
3. Run the main file
4. Enjoy


### Requirements

- Python 3.x
- Flask
- SQLAlchemy
- A computer 
- A brain
